# Otimização de Calendário — Lucas Meninelli

## Diagnóstico
- **Reuniões:** ocupam ~65% do tempo disponível
- **Blocos de foco:** menos de 2h/dia em média
- **Problema principal:** reuniões fragmentam blocos de trabalho profundo
- **Padrão:** maioria das reuniões sem pauta definida com antecedência

## Recomendações aplicadas

### Estrutura semanal proposta
| Período | Uso |
|---|---|
| Seg manhã | Planejamento semanal + revisão pipeline |
| Ter/Qui | Reuniões externas (clientes, demos) |
| Qua | Deep work (sem reuniões até 12h) |
| Sex manhã | Review + follow-ups |
| Sex tarde | Projetos estratégicos / desenvolvimento |

### Regras implementadas
- Reuniões internas: máximo 30 min, pauta obrigatória
- No-meeting Wednesday mornings
- Buffer de 15 min entre reuniões consecutivas
- Blocos de foco protegidos no calendário (não abrir para agendamento)

## Ferramentas
- Google Calendar com blocos de foco marcados como "ocupado"
- Integração com HubSpot para reuniões de vendas
- Calendly configurado apenas para horários aprovados

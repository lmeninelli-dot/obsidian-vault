# Make.com — Plataforma de Automação 2026

## 📌 O que é
Make.com (ex-Integromat) é uma plataforma visual de automação no-code que conecta apps e serviços via **Scenarios** — workflows visuais em canvas onde cada módulo representa uma ação.

> Posicionamento 2026: "Visual automation that thinks, decides, and adapts with you."

---

## 🆕 Principais novidades 2026

### AI Agents (lançado abril 2025, consolidado em 2026)
- Agentes autônomos com LLMs que tomam decisões dentro dos scenarios
- Não precisam de instruções passo-a-passo — raciocinam para atingir objetivos
- Capacidades: "Request Anything" (linguagem natural), análise de sentimento, categorização de texto

### MCP Server (cloud-hosted)
- Acesse seus scenarios de qualquer interface de IA
- Sem servidor para gerenciar — hospedado pelo Make
- Integração direta com Claude, ChatGPT, Cursor e outros

### Módulos de IA prontos (no-code)
- Categorização, sumarização, geração de texto sem precisar configurar prompts
- 400+ integrações com apps de IA: Claude, ChatGPT, Gemini, Grok e outros

### Novo modelo de créditos (agosto 2026)
- Transição de billing por operações para **sistema de créditos**
- A maioria das ações ainda consome 1 crédito
- Módulos de IA avançados podem consumir mais

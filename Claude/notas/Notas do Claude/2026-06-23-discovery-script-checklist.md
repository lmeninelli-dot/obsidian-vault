# Discovery Script + Pre-Demo Checklist

## Contexto
Gerado a partir do relatório de Skills Gap do time de vendas da Workise (PDF analisado em Jun 23).

## Discovery Script — Estrutura
1. **Abertura** — rapport + agenda da call
2. **Situação atual** — como gerenciam projetos/processos hoje
3. **Dores** — onde perdem tempo, onde há retrabalho
4. **Impacto** — quanto isso custa (tempo, dinheiro, pessoas)
5. **Visão** — como seria o cenário ideal
6. **Fit check** — validar se monday.com é a solução certa
7. **Próximos passos** — sempre com data definida

## Pre-Demo Checklist
- [ ] Pesquisar empresa (setor, tamanho, stack atual)
- [ ] Identificar dores mencionadas na discovery
- [ ] Preparar demo personalizado (não genérico)
- [ ] Testar ambiente de demo antes
- [ ] Preparar board/workspace específico para o caso de uso
- [ ] Identificar quem vai estar na call (decisores?)
- [ ] Preparar ROI estimado baseado no setor

## Skills Gap identificados (relatório)
- Baixa taxa de conversão discovery → demo (pergunta pouco sobre impacto financeiro)
- Demos genéricas sem personalização para o setor
- Follow-up inconsistente pós-demo

## Onboarding Board criado
Board monday.com para onboarding de novos vendedores baseado nos gaps identificados.
